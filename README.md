

# VR Payment plentymarkets
This repository contains the plentymarkets extension that enables to process payments with [VR Payment](https://www.vr-payment.de//).

###### To use this extension, a [VR Payment](https://www.vr-payment.de//) account is required.

## Requirements

* [plentymarkets 7](https://www.plentymarkets.eu/)
* [IO](https://marketplace.plentymarkets.com/plugins/channels/online-shops/io_4696)
* [Ceres](https://marketplace.plentymarkets.com/plugins/channels/online-shops/ceres_4697)
